About this product

  Entities Directory: a directory of entities with or without web site or email.
    
Installation

  * Tested on Plone 3.1.

  * Copy the collective.entitiesdirectory folder to your buildout src folder and add the egg to your buildout.cfg eggs and zcml sections.
  
  * Restart Zope
  
  * Go to Plone configuration menu "Add/Remove product", select the product and install it.
  
  * Have fun!

Testing

  * run 'bin/buildout -s collective.entitiesdirectory'
